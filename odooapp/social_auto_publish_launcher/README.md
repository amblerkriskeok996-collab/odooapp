# Social Auto Publish Launcher

Standalone Odoo 19 app that adds a visible backend launcher entry for the
existing one-click publish application.

Current behavior:
- shows up as a standalone app in the Odoo app launcher
- exposes a backend child menu named `发布中心`
- opens `http://localhost:5173/` in a new browser tab

This phase still does not add native Odoo business models, controllers, or a
Social Marketing bridge module.
